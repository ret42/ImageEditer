# Image Editer

Simple tool in python for convert image format, edit etc...
lib used: Pillow

## Features:
- Fix corrupted image (support mp4) /!\ Only for bytes
- Change format
- Video to gif
- Mirror
- Crop
- Rotate
- Filters
