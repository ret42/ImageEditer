# Image Editer

Simple tool in python for convert image format, edit etc...<br>
lib used: Pillow, transforms

## Features:
- Fix corrupted image /!\ Only for bytes (support mp4)
- Change format
- Video to gif
- Mirror
- Crop
- Rotate
- Image to ASCII
- Filters
  - Autocontrast
  - Black And White
  - Invert
  - Tint
  - Saturation
