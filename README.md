# Image Editer

Simple tool in python for convert image format, edit etc...<br>
lib used: Pillow

## Features:
- Convert
  - PNG
  - JPG
  - WEBP
  - GIF
- Fix Corrupted Image (/!\ Bytes only)
- Edit Image
  - Mirror
  - Flip
  - Rotate
  - Crop
  - Resize
  - Image To ASCII
  - Split
- Filters
  - Autocontrast
  - Black and white
  - Invert
  - Tint
  - Saturation
  - Blur
  - Brightness
  - Contrast
  - Merge
